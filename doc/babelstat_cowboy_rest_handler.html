Description

Restful interface for the babelstat api. You query the database using the query and filter ex. http://localhost/babelstat?category="category"&sub_category="sub_category"& subject="subject"&series_category="series_category"&title="title"& frequency="frequency"&metric="metric"&scale="scale"

You can add the optional &from_date="iso formatted date"&to_date="iso formatted date"

The response will tell you if any of the parameters are missing.
