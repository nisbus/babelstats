Description

A websocket handler for the babelstat service. Takes a json document for calling for data in the form: {"query": {"category" : "cat", "sub_category" : "sub", "subject" : "subject", "series_category" : "series_category", "title": "title", "frequency":"frequency", "scale":"scale", "metric":"metric", "from_date": "optional", "to_date": "optional"} }

This is mainly for fun and has no real use cases, use the rest interface instead :)
